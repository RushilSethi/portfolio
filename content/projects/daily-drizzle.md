## Description

**DailyDrizzle** is a **real-time weather application** that allows users to **search for current weather conditions** manually or through **GPS-based location detection**. It provides a **3-day forecast**, real-time updates on **temperature, humidity, wind speed, and AQI**, and features an **intuitive UI** with a **responsive design** optimized for all devices.
